# JUCE Programming Skill

This is a Claude Code skill for expert guidance on JUCE C++ audio framework development.

## What This Skill Provides

Comprehensive assistance with:
- Audio plugin development (VST3, AU, AAX, LV2)
- GUI application development
- CMake build configuration
- Audio processing and DSP
- JUCE module usage and best practices
- Cross-platform development (Windows, macOS, Linux, iOS)

## How It Works

This skill uses **progressive loading** - Claude reads the main `SKILL.md` file first, then accesses supporting reference files only when needed for deeper information.

### Main File
- **SKILL.md**: Core guidance, best practices, and workflow instructions

### Supporting Reference Files (Progressive Loading)
- **modules-reference.md**: Complete guide to all 23 JUCE modules
- **cmake-options.md**: Comprehensive CMake API reference
- **audio-processing-patterns.md**: Audio processing techniques and patterns
- **gui-patterns.md**: GUI development patterns and best practices
- **plugin-formats.md**: Details on VST3, AU, AAX, LV2, and other formats

## When This Skill Activates

Claude will automatically use this skill when you:
- Work on JUCE projects
